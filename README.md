# SPOTIFY-USER-BEHAVIOUR-ANALYSIS-DASHBOARD

Overview
This project is a Power BI dashboard that analyzes Spotify user behavior, listening patterns, and subscription insights. It helps in understanding how users interact with music, ads, and features.

Key Insights
Average listening hours per week
Skips per day by favorite genre
Subscription type & status distribution
Ad interaction vs conversion
User engagement based on inactivity
Most liked songs & genres

Visualizations Used
Bar Charts → Genre vs Skips
Line Charts → Listening trends
Pie/Donut Charts → Subscription types
Funnel Chart → Ad conversion process
Ribbon Chart → Listening vs skipping trends
Cards → KPI metrics

Tools Used
Power BI Desktop
Data modeling & DAX
Data visualization techniques

Purpose
This dashboard helps:
Understand user engagement
Improve music recommendations
Analyze subscription behavior
Optimize ad strategies
